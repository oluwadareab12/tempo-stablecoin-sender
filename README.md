# Tempo Testnet Stablecoin Sender

A simple frontend dApp built on **Tempo Testnet** that allows users to:

- Connect their MetaMask wallet
- View their stablecoin balance (AlphaUSD)
- Send stablecoins to another address

This project is intended as a **builder demo** and learning project.
